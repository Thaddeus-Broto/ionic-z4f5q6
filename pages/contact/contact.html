ABOUT

Reﾌ《oudre is a markdown calculator that enables you to see how much you are saving by showing you the difference between the original price and the lowered selling price. Results will be shown in raw digits as well as percentage form.

Reﾌ《oudre was encoded through Stackblitz using HTML, Javascript, and CSS. This application makes use of elements sourced from Github and Canva.

How to use the App:

Enter the original price of the product, service or commodity in the box labeled 'Original Price'. Afterwards, enter the selling price of that same product, service, or commodity in the box labeled 'Selling Price'. Finally, press compute and you will get how much you were able to save and the percentage of the markdown rate.

How Reﾌ《oudre Works:

The Reﾌ《oudre app was created using coding from StackBlitz. The app utilizes HTML for display, Javascript for the variables and CSS for the design and aesthetics of the interface. The application works when two amounts, the original price and selling price, are entered. The codes use a basic formula that subtracts the 'Original Price' from the 'Selling Price' to get the 'Savings'. The Marketdown Rate is computed when 'Savings' and 'Selling Price' are divided and then multiplied by 100.
